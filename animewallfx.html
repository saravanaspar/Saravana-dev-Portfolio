AnimeWallFX

A high-performance media gallery engineered to load thousands of HD assets without browser lag.

React.js Supabase Lazy Loading

The Challenge

Standard image galleries suffer from layout shifts (CLS) and slow loading times when handling HD wallpapers. The goal was to build a site that feels "instant" on mobile networks.

The Solution

Implemented a virtualized grid system combined with aggressive image optimization via Supabase, reducing First Contentful Paint (FCP) to under 0.8s.
